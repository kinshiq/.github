# Kinshiq

Kinshiq is a machine-first system for representing and reasoning about
relationships between individuals.

The project develops specifications, tools, and reference implementations for
machine-friendly relationship encoding and analysis.

## Project

The Kinshiq project develops:

- **Specifications** defining the Kinshiq language
- **Reference implementations** of the specification
- **Tools** for relationship analysis and computation

Specifications are available in the [spec repository](https://github.com/kinshiq/spec).

Website: https://kinshiq.org
